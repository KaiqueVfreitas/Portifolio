Sobre Mim

Minha jornada na tecnologia começou em 2018 com o professor Gustavo Guanabara e sua plataforma Curso em Vídeo, quando eu tinha 13 anos. Iniciei com o curso de lógica de algoritmos aprendendo Portugol, seguido por Python e Java, incluindo programação orientada a objetos, além de Git e GitHub.

Em 2020, busquei cursos profissionalizantes mais abrangentes, o que me levou à Fundação Bradesco, onde me formei em Office 2016 (nível intermediário) e Power BI (nível básico).

No ano de 2021, decidi explorar o mundo da Web através da OneBitCode, onde aprendi tecnologias como Bulma (Framework CSS), MongoDB, Next.js, HTML, CSS e JavaScript, TypeScript, Node.js, SASS/SCSS, Git e GitHub. Além de estudar a parte teórica, apliquei essas tecnologias em projetos práticos como replicar interfaces do Nubank e Netflix, desenvolver um controlador de música com consumo de API e criar uma plataforma de cursos EAD hospedada.

Antes de ingressar na Escola Técnica Estadual de São Paulo - ETEC em 2023, fiz um curso de Microsoft Azure no Senai, obtendo certificados internacionais da Microsoft em Inteligência Artificial: Microsoft Azure Fundamentals e Microsoft Marching Learning. No curso de Marching Learning, aprendi sobre treinamento visual (estado, tempo e cor), treinamento sonoro (frequência, distância e altura) e técnicas para garantir precisão textual.

Atualmente, estou cursando Análise de Dados aos sábados na Hashtag Treinamentos, onde aprimoro minhas habilidades em Python para criação de dashboards utilizando SQL, Excel, CSV e TXT como bases. Além disso, estou explorando tecnologias como VBA e Power Apps, desenvolvendo a mentalidade necessária para identificar e solucionar problemas de forma eficaz.

Por fim, na ETEC, adquiri conhecimento em diversas tecnologias e áreas, como MySQL, Microsoft SQL Server, PHP com XAMPP para desenvolvimento web, Android Studio para criação de aplicativos com banco SQLite e consumo de API, Java (incluindo Programação Orientada a Objetos), C# com Windows Forms e ASP.NET MVC para CRUD em MySQL. Também utilizei HTML, CSS e JavaScript em trabalhos como um TCM e um TCC, além de explorar CorelDRAW, Canvas e APIs de ASP.NET para Web. Em sistemas embarcados, utilizei C++ para criar um sistema de cofre com Arduino e Autodesk Tinkercad.
